
# Moovies App

Technologies
============
1. UrlSession
2. CocoaPods
3. MVVM Pattern
4. UI/UX
5. KingFisher

Features
========
+ Get All datas from Api(omdb).
+ Search as you need.
+ Read details.
+ Searchig indicator.
+ No data screen when no movie found


## Installation

Clone, Download the Repository

```bash
pod install 
and run with Xcode
```
